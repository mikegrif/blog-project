Learning HTML

Mike G. | 06-20-2020

Mike G is a Software Engineer who works in the Telecommunications Industry. He lives in the southeastern part of the United States, and he loves to learn about new technologies.

HTML stands for the Hyper Text Markup Language. HTML is considered the "lingua franca" of the web. It is the language that determines the structure of layout of a webpage.

HTML has elements that form the building blocks of a webpage. To visually appealing webpages, HTML is used with other technologies such as Cascading Style Sheets (CSS) and JavasScript.
